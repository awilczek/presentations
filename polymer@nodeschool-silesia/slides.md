# Web Components

- Custom elements
```
<my-tag></my-tag>
```
- Imports
```
<link rel="import" href="my-tags.html">
```
- Shadow DOM/CSS
![Shadow DOM](images/shadowdom.gif)
- Templates
```
< template >
      <img src="large.gif"> // this will not fire any request (yet)
< /template >
```
